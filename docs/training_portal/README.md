# ATLC Training Portal

Static training portal for GitHub Pages.

Publish source: `docs/`

Expected URL:

```text
https://<username>.github.io/<repo-name>/training_portal/
```

Replace placeholders:

- Google Drive video links in `sections/*.html`
- PDF slides in `slides/`
- Overleaf ZIP in `downloads/`
# quizzes folder

This package contains regenerated HTML quizzes for Section 01 to Section 06.

Files:

- quizzes/quiz_section03.html
